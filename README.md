# promicro-gamepad

## Depends on

* [MHeironimus/ArduinoJoystickLibrary](https://github.com/MHeironimus/ArduinoJoystickLibrary)

## Pin assign

Pin assign for [Pro Micro](https://www.sparkfun.com/products/12640)(or compatible) board.

| on board   | Digital in  | GamePad     |
|:-----------|:------------|:------------|
| A0         | 18          | D-pad Up    |
| A1         | 19          | D-pad Right |
| A2         | 20          | D-pad Down  |
| A3         | 21          | D-pad Left  |
| 2          | 2           | Button 1    |
| 3          | 3           | Button 2    |
| 4          | 4           | Button 3    |
| 5          | 5           | Button 4    |
| 6          | 6           | Button 5    |
| 7          | 7           | Button 6    |
